<p>mangoOS is a 64-bit x86-64 operating system built using C. It is powered by the Limine bootloader and utilizes the CMake build system. Its long-term direction is unknown for now, and the project may continue to grow or conclude as our goals evolve. Nonetheless, we are thrilled to learn about opearting systems and see where it leads us.</p>

## Prerequisites

- x86_64-elf cross-compiler
- CMake
- GNU Make
- QEMU
- xorriso
- Limine bootloader 

> **Note:** You can build the cross-compiler by following the [OSDev GCC Cross-Compiler Guide](https://wiki.osdev.org/GCC_Cross-Compiler) or watching this [YouTube tutorial](https://www.youtube.com/watch?v=zBorEnGhPv8).
